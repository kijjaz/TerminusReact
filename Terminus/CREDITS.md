# Terminus Project Credits & References

This project draws inspiration and technical patterns from the following open-source projects:

## Roguelike Mechanics & World Generation
*   **Angband** ([GitHub](https://github.com/angband/angband)): Inspiration for ANSI-based terrain data, room templates, and dungeon profiles. A legendary roguelike maintained by the community.

## Multiplayer Architecture
*   **ModernUO** ([GitHub](https://github.com/modernuo/ModernUO)): Study of high-performance, asynchronous multiplayer server design and persistent world synchronization.

## Assets & Textures (Previous Phase)
*   **Luanti / Minetest** ([GitHub](https://github.com/minetest/minetest)): Source of 16x16 pixel textures utilized in the Minerio gameplay prototype.

## Core Technology
*   **Socket.io**: Real-time bidirectional event-based communication.
*   **Express**: Minimal and flexible Node.js web application framework.
*   **VT323 Font**: Google Fonts (Pixel-art terminal aesthetic).
